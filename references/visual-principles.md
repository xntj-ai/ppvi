# 视觉原则：克制优雅

## 核心理念

设计是减法过程。每增加一个视觉元素，都在消耗用户的注意力预算。好设计的标志不是"加了什么让它好看"，而是"什么都不能再删了"。

## 四象限判断法

面对任何视觉元素，问两个问题：
1. 它传达信息吗？（功能性）
2. 它建立氛围吗？（情感性）

| | 传达信息 ✓ | 不传达信息 ✗ |
|---|---|---|
| **建立氛围 ✓** | 保留并强化 | 谨慎保留，控制量 |
| **不建立氛围 ✗** | 保留但简化表达 | **删除** |

## 克制不等于简陋

克制是选择后的精致，不是偷懒后的空白。

**克制的**:
- 一个颜色用到极致，用明度/饱和度变化创造层次
- 统一的 8px 圆角，偶尔用 50% 圆角做强调
- 一种阴影深度覆盖 90% 场景，极少数用深阴影做提升

**简陋的**:
- 全部灰色没有层次
- 没有圆角（或随机圆角）
- 没有阴影也没有其他深度手段

## 装饰的使用规则

1. **装饰必须有节奏** — 不是"这里空，加个图标"，而是有规律的视觉节奏
2. **一种装饰手法用到底** — 选了渐变就不要同时用阴影做同样的事
3. **装饰不抢内容** — opacity ≤ 0.1 的背景纹理 OK，opacity 0.5 的花纹不行
4. **少即是多的量化标准** — 一个页面的装饰性元素 ≤ 3 种类型

## 实战检查清单

设计完成后，逐项自检：

- [ ] 遮住任意一个装饰元素，设计是否仍然完整？如果是 → 删掉它
- [ ] 所有圆角值是否来自统一的设计 token？（如 4/8/16/full）
- [ ] 颜色数量是否 ≤ 5（含灰阶）？
- [ ] 是否存在"看起来差不多但又不完全一样"的重复样式？→ 统一
- [ ] 首屏信息层级是否在 0.5 秒内可识别？
